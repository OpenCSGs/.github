## OpenCSG 开放传神
OpenCSG中 C 代表 Converged resources，意思是整合和充分利用好混合资源；S 代表 Software refined，也就是软件被大模型重塑；G 代表 Generative LM，是大众化和民主化的生成式大模型。
OpenCSG的愿景是让每个行业、每个公司、每个人都拥有自己的模型。为此，我们秉持开源开放的原则，将OpenCSG的多个软件开源到社区，欢迎使用、反馈和参与共建。
